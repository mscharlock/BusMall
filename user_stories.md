# BusMall User Stories
---
## Developers
* I want to write clean code that is modularized.
* I want to declare variables at the top of my code and comment major sections so it is easier for other developers to read.
* I want to log the clicks of each user, so I can tell which product they like best.
* I want to display the results of a user's clicks in a chart and save it to local storage.
## Marketing
* I want to find out which items users like the most, so I can stock my catalog with the right products.
* I want to make sure I know what users are clicking on the most in an easy-to-read format.
* I want to make sure the look/feel/experience for users is the best possible.
* I want to make money from Bus Mall!
## Users
* I want to easily select which items I like best.
* I want to see the results from this test.
* I want a cohesive look/feel/experience while doing this test.  
* I may want to purchase something from Bus Mall after this test.
